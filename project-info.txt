🐱🦜 ПРОЕКТ: МурЛэнд Спа - Спа-центр с котиками и попугайчиками
================================================================

📍 ЖИВОЙ САЙТ:
https://practice-task-omega.vercel.app/

💻 ИСХОДНЫЙ КОД:
https://github.com/KriiU/practice-task

📋 ОПИСАНИЕ:
Интерактивное веб-приложение на React для спа-центра "МурЛэнд Спа", 
где можно заказать уникальные процедуры с участием пушистых терапевтов - 
котиков и попугайчиков.

✨ ОСНОВНЫЕ ФУНКЦИИ:
- 🏠 Главная страница с описанием услуг
- 📦 Каталог спа-услуг (котики и попугайчики)
- 👤 Личный кабинет с историей бронирований
- 🎨 Современный UI с Bootstrap стилями
- 📱 Адаптивный дизайн для всех устройств

🛠 ТЕХНОЛОГИИ:
- React.js
- Bootstrap 5
- JavaScript ES6+
- CSS3
